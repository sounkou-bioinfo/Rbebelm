# Start an interactive Rbebelm console agent

Start an interactive Rbebelm console agent

## Usage

``` r
bebel_r_agent_console(session, prompt = "bebel> ", max_steps = 4L)
```

## Arguments

- session:

  A `bebelRAgent`.

- prompt:

  Prompt string.

- max_steps:

  Maximum assistant/tool iterations per user prompt.

## Value

Invisibly returns `session`.
