# Open an agent session JSONL file

Open an agent session JSONL file

## Usage

``` r
bebel_session_open(path, session_dir = NULL, cwd = NULL)
```

## Arguments

- path:

  Session JSONL file.

- session_dir:

  Optional session directory for future derived sessions.

- cwd:

  Optional working-directory override.

## Value

An `bebelSession` object.
