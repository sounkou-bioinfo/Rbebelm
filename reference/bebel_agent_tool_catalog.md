# Describe an Rbebelm agent tool catalog

Describe an Rbebelm agent tool catalog

## Usage

``` r
bebel_agent_tool_catalog(tools)
```

## Arguments

- tools:

  A tool list accepted by
  [`bebel_r_agent()`](https://sounkou-bioinfo.github.io/Rbebelm/reference/bebel_r_agent.md).

## Value

A data frame with tool names and descriptions.
